# 📱 Household Expense Tracker App

A clean and scalable mobile application built with React Native (Expo) to track daily household consumption and expenses.

---

## 🚀 Features

* ✅ Add custom household items (milk, vegetables, etc.)
* ✅ Track daily usage with quantity
* ✅ Select custom dates for entries
* ✅ Edit & delete entries
* ✅ Price per unit tracking (₹)
* ✅ Automatic expense calculation
* ✅ Monthly expense summary
* ✅ Visual charts (bar graph)

---

## 🧠 Tech Stack

* React Native (Expo)
* Expo Router
* Zustand (state management)
* AsyncStorage (local persistence)
* react-native-chart-kit (charts)

---

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/household-expense-tracker-app.git
cd household-expense-tracker-app
npm install
npx expo start
```
